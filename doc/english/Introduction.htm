Introduction

Présentation

Simple Web Script is is a PHP-like. He is free and open source

Langage de base

Simple Web Script language is strict. One each line (except empty line and commande line) you have a command.
But, Simple Web Script is not case sensitive and you can have same name for variable and function.

WARNING :
You can have only ONE instrution per line.

Limitations

Simple Web Script is not a big programmation language ! He is just easy keyboard programming.

Commande Line

-debug : active debugger
-file <file_name>: file to read
-params <arg1 arg2 arg3 ...>: arguments

How to compile

For compile Simple Web Script, you must have Delphi 6 or Lazarus 0.9.22 (or greater) and Free Pascal Compiler 2.0 (or greater). Open SimpleWebScript.dpr and compile it.

To configure SimpleWebScript edit config.inc for général configuration.

To configure where ini file is, edit path_ini.inc.
