Introduction

Présentation

Simple Web Script est un logiciel de type PHP. C'est un logiciel gratuit et open source.

Langage de base

Le langage d'Simple Web Script est par certain coté très strict. Il doit y avoir obligatoirement une commande sur chaque ligne (excepter les lignes vide et les lignes de commentaire).
Toutefois, Simple Web Script ne fait pas la distinction entre majuscule et minuscule. Une variable peut avoir le même nom qu'une fonction ou qu'un label.

Limitations

Simple Web Script ne pas vocation à être un langage de programmation puissant. Il propose simplement des facilités d'utisation.

Paramètres de lignes de commandes

-debug : active ledebuggeur
-file <file_name>: fichier à lire
-params <arg1 arg2 arg3 ...>: arguments

Comment compiler

Pour complilet Simple Web Script vous devez avoir Delphi 6 ou Lazarus 0.9.24 beta (ou suppérieur) et Free Pascal Compiler 2.0 (ou suppérieur). Ouvrez SimpleWebScript.dpr (sous Delphi) ou SimpleWebScript_optimisé.lpi (sous Lazarus) et comilez.

Pour configurer SimpleWebScript éditer le fichier config.inc.

Pour configurer l'emplacement du fichier ini de configuration (sws.ini), éditer path_ini.inc.
