OpenAutoKey à un debbuger intégrer en ligne de commande. Pour l'activer, il suffit de lançer le script avec le paramètre -debug.
Le debugger va alors exécuter le script ligne par ligne. Les commandes suivantes sont diponibles :
next : saut à l'instruction suivante
skip : n'exécute pas la prochaine ligne
setvar $mavariable mavaleur : change ou ajoute une variable
getvar $mavariable : retourne la valeur de la variable
showvar : affiches le liste des variables disponibles
showlabel : affiche la liste des labels disponibles et leur numéro de ligne associés
showproc : affiche la liste des procedure

Il y a un autre argument intéressant c'est -nowarning. Cela désactive les messages d'alertes (e.g. si une variable n'existe pas)
